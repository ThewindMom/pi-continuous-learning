# Harness principles

- Pi history remains the only transcript source of truth.
- Foreground work stays local, bounded, and model-free.
- Repeated evidence beats one-off anecdotes.
- Candidates are isolated from approved memories until canaries help.
- Harmful attributed outcomes disable rules rather than silently preserving them.
- State transitions are idempotent and cursor advancement is committed with
  the resulting lifecycle changes.
- Deployment targets receive only mature approved knowledge.
- Complexity is removed unless it improves future runs without slowing current
  runs.
